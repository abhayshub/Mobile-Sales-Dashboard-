
📱 Mobile Store Sales Analytics Dashboard

📌 Project Overview

An interactive Power BI dashboard built to analyze mobile store sales performance, customer behavior, payment methods, and brand-wise insights across different cities and months.



🎯 Purpose

The dashboard helps businesses track KPIs, identify sales trends, monitor customer preferences, and make data-driven decisions using interactive visualizations.




🛠️ Tech Stack

📊 Power BI Desktop

📂 Power Query

🧠 DAX

📝 Data Modeling

📍 Map Visualizations





📁 Data Source

Sample Mobile Store Sales Dataset containing:

Mobile Brands & Models

Sales & Transactions

Customer Ratings

Payment Methods

City-wise & Monthly Sales Data



---

✨ Dashboard Features

📈 Total Sales, Transactions & Quantity KPIs

🌍 City-wise Sales Analysis using Maps

📅 Monthly Sales Trend Analysis

📊 Brand-wise Performance Comparison

💳 Payment Method Distribution

⭐ Customer Ratings Insights

📆 Sales Analysis by Day Name

🎛️ Interactive Month Filters & Slicers





📊 Key Insights

Apple and Samsung generated the highest sales.

UPI and Card payments were the most preferred payment methods.

Sales performance varied across months and weekdays.

Major cities contributed significantly to total revenue.
