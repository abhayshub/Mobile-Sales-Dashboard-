📱 Mobile Store Sales Analytics Dashboard

1. Project Title / Headline

📊 Mobile Store Sales Analytics Dashboard
An interactive Power BI dashboard designed to analyze mobile store sales performance, customer behavior, transaction trends, and brand-wise insights across different cities and time periods.


2. Short Description / Purpose

The Mobile Store Sales Analytics Dashboard is a dynamic and visually interactive Power BI report created to monitor and analyze mobile phone sales data.
This dashboard helps businesses, analysts, and decision-makers track KPIs such as total sales, transactions, customer ratings, payment methods, and product performance to gain valuable business insights and improve strategic decision-making.



3. Tech Stack

The dashboard was built using the following tools and technologies:
📊 Power BI Desktop – Main platform used for data visualization and dashboard creation.
📂 Power Query – Used for data cleaning, transformation, and preprocessing.
🧠 DAX (Data Analysis Expressions) – Used for calculated measures, KPIs, and dynamic analytics.
📝 Data Modeling – Relationships between tables were created to enable filtering and interactive reporting.
📍  Map Visualization – Used for city-wise sales analysis across India.
📁 File Format – .pbix for development and .png for dashboard preview/screenshots.


4. Data Source

Source: Sample Mobile Store Sales Dataset

The dataset contains mobile sales transaction records including:
Mobile Brands & Models
Sales Amount
Quantity Sold
Customer Ratings
Payment Methods
Transaction Details
City-wise Sales Data
Monthly Sales Trends


5. Features / Highlights

• Business Problem

Mobile retail businesses generate massive amounts of sales data daily, but understanding sales performance, customer purchasing behavior, and product trends can be difficult using raw spreadsheets or traditional reports.
Businesses often struggle to answer questions like:
Which mobile brand generates the highest revenue?
Which months perform best in sales?
What payment methods are most preferred by customers?
Which cities contribute the highest sales volume?
How do customer ratings impact product performance?


Without a centralized dashboard, decision-making becomes slower and less effective.



• Goal of the Dashboard

The goal of this dashboard is to provide an interactive and easy-to-understand business intelligence solution that:
Tracks overall sales performance and KPIs
Identifies top-performing brands and products
Monitors monthly and daily sales trends
Analyzes customer purchasing behavior
Helps businesses make data-driven decisions

Useful for:

Helps identify:
Best-performing sales days
Customer shopping patterns
Weekly sales behavior


• Business Impact & Insights

📈 Improved Decision-Making
Businesses can quickly identify top-performing products, brands, and locations using real-time visual insights.

🛍️ Better Customer Understanding
Customer ratings and payment trends help businesses improve customer experience and payment convenience.

📊 Sales Optimization
Monthly and daily trend analysis enables better sales forecasting and inventory planning.

🌍 Regional Market Insights
The geographical analysis helps identify high-potential cities and target markets for expansion.

💡 Strategic Growth
Management teams can use these insights for marketing campaigns, pricing strategies, and business planning.
